YOLO_v3-分割法結果

### 主要特性

YOLO的缺點在於對於範圍廣闊的目標辨識成效不彰，對於小範圍的檢測更加優秀， 由於這機制的關係，我使用了分割法將照片的範圍縮小，這將有利於檢測的成效，這裡做了兩個版本，而碩士論文為使用keras文件作為測試數據

|   | keras文件  | tensorflow文件  |
| :------------: | :------------: | :------------: |
|  架構 | 上採樣時沒有加上另外兩個網格特徵  | 論文架構  |
|  優點 |   近距離極佳|   中遠距離佳|
|  缺點 |  中遠距離不好|   近距離不好|
